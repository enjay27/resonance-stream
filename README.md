# 🛰️ Resonance Stream

**Blue Protocol: Star Resonance (BPSR) Real-time Packet-Sniffing Translator**

---

## 📖 소개 (Introduction)
![normal-mode](https://github.com/user-attachments/assets/a78ff1c9-ddc7-4c62-8104-162e397f8512)


![compact-mode](https://github.com/user-attachments/assets/733550a1-f05e-49b4-9b5a-2ab83756fdb7)

**Resonance Stream**은 블루 프로토콜 게임 서버의 네트워크 패킷을 스니핑하여 대화 내용을 추출하고 번역합니다. 이를 통해 UI 에 채팅 내용을 그대로 가져올 수 있으며, 닉네임 검색, 대화 내용 검색, 키워드 알림 등의 다양한 기능을 제공합니다. 또한 번역이 필요한 일본어 채팅의 경우에는 AI 모델을 탑재하여 실시간 번역을 제공합니다.

### ✨ 핵심 기능 (Key Features)

* **채팅 서버 패킷 스니핑**: Raw Socket 을 이용하여 블루 프로토콜 게임 서버에서 보낸 채팅을 가져옵니다.
* **AI 번역 엔진**: 일본어 채팅 데이터로 파인튜닝된 Qwen3-4B 모델을 사용하여 실제 게임에 사용되는 언어로 번역됩니다.
* **닉네임 번역**: 일본어 닉네임의 읽는 방법을 닉네임 옆에 괄호를 열어 추가합니다.
* **다양한 디바이스 지원**: 사용자 하드웨어 사양에 맞게 CPU / GPU 모드, GPU Off Load Tier (Low, Middle, High, Very High) 로 설정이 가능하며, Vulkan 을 사용하여 제조사에 관계 없이 GPU를 사용할 수 있습니다.
* **다양한 UI 기능**: 게임 내의 채팅 기능이 제공하지 못하는 다양한 기능을 제공합니다.

## 🛠️ 설치 및 요구 사항 (Setup & Requirements)

### 필수 구성 요소

1. **관리자 권한**: 패킷 감지를 위해 프로그램 실행 시 관리자 권한이 반드시 필요합니다.
2. **네트워크 방화벽 설정**: 이 프로그램은 패킷 감지를 위해 네트워크 방화벽 설정이 필요합니다. 현 버전에서는 사용자의 동의 없이 방화벽 규칙을 설정하며, 생성된 규칙은 프로그램 종료 시 제거됩니다. 다음 버전에서는 사용자의 동의를 받아 규칙을 추가하는 것으로 변경될 예정입니다.

### 설치 방법

1. 최신 [Release] 섹션에서 `resonance-stream.exe` 파일을 다운로드합니다.
2. `resonance-stream.exe`를 **관리자 권한**으로 실행합니다.

## 🗑️ 삭제 안내 (Uninstall Guide)

**Resonance Stream**의 변역 기능을 사용한 경우, 번역을 위해 약 **2.3GB**의 데이터를 사용합니다. 프로그램을 삭제하실 때 다음 사항을 확인해 주세요.

1. **앱 데이터 폴더 확인**: 프로그램의 설정 창 맨 밑의 App Data 폴더 열기 버튼을 눌러 프로그램 추가 설치 파일 폴더를 엽니다.
2. **데이터 삭제**: 폴더 내의 모든 설치 파일을 제거합니다.
3. **수동 확인 (선택 사항)**: `%APPDATA%\com.enjay.bpsr.resonance-stream` 폴더를 직접 삭제하시면 됩니다.

## 🚀 사용법 (Usage)

1. 프로그램 최초 실행 시 최초설정 창이 나오며, 번역 사용 여부, CPU/GPU 모드 설정을 할 수 있습니다. 이 설정은 프로그램 실행 후 설정에서 바꿀 수 있습니다.
2. 게임(블루 프로토콜)을 실행하고 서버에 접속하면 자동으로 패킷 감지가 시작됩니다.
3. 만약 게임을 실행하였는데도 게임의 채팅 데이터를 받아오지 못하는 경우에는 프로그램 맨 위 타이틀바에서 초록색의 ```SNIFFER ON``` 이 되었는지 확인해주세요. 빨간색인 경우, 클릭하면 실패 원인을 확인할 수 있습니다.
4. 번역 기능을 켠 경우, ```번역 ON``` 이 되었는지 확인해주세요.

## 📚 사용자 사전 추가

1. **앱 데이터 폴더 열기**: 설정 창 맨 밑의 ```앱 데이터 폴더 열기``` 버튼을 눌러 사용자 사전이 저장된 폴더를 엽니다.
2. **사용자 사전 파일 열기** : 해당 폴더의 custom_dict.json 를 열어 포맷에 맞게 변경할 내용을 적용합니다.
3. **사용자 사전 초기화**: 만약 번역 오류가 발생하는 경우, 설정창의 사전 동기화 버튼을 눌러 최초 실행시의 사용자 사전으로 초기화 할 수 있습니다.

## 📜 버전 히스토리 (Changelog)

### v0.3.1 (2026-03-03) - 정식 출시

- **번역 엔진 변경**: 기존의 파이썬 라이브러리 번역에서 파이썬 의존성을 제거하고 번역 엔진으로 Vulkan 드라이버를 지원하는 llama-server 를 사용하여 모든 종류의 GPU 에서 번역 기능을 GPU 로 사용할 수 있도록 변경하였습니다.
- **번역 모델 변경**: CTranslate2 의 NLLB-200 모델에서 파인튜닝 된 Qwen3-4B 모델로 변경하여 게임 내의 일본어 채팅을 자연스러운 한국어로 번역할 수 있게 합니다.
- **패킷 분석 의존성 제거**: WinDivert 에서 Raw Socket 으로 변경하여 의존성 파일이나 추가 설치 파일 없이 단일 실행 파일로 프로그램을 실행할 수 있도록 합니다.
- **닉네임 관리**: 일본어로 된 닉네임의 읽는 방법을 로마자로 추가하고 닉네임을 캐싱하여 닉네임이 채팅에 나오는 경우 번역되지 않도록 합니다.
- **디버그 시스템**: 네트워크 드라이버 빛 번역 기능 동작 확인을 위한 시스템 로그(Debug Mode)를 지원합니다.
